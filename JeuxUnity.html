World of Crystals

J'ai réalisé les dessins en 2D d'un petit jeu codé en C++ sur Visual Studio. J'ai fait les dessins sur le logiciel Krita. Le jeu est un labyrinthe dans le style du jeu Pacman dans lequel le joueur doit récupérer des objets en évitant de se faire toucher par des monstres. Il doit ensuite atteindre un point du labyrinthe pour finir le jeu.

Extasia

Extasia est un jeu très coloré codé en C#. J'ai réalisé ce jeu dans son intégralité à l'aide du moteur de jeu Unity. Le code a été fait sur Visual Studio. Les assets et UI ont été dessiné sur Krita pour être ensuite implémentés dans Unity.

Dans Extasia, le jeu au concept surprenant, on contrôle les mouvements d'un personnage ayant une addiction pour les drogues. Celles-ci sont représentées par des friandises. Le joueur doit dans ce jeu récupérer un maximum de bonbons tout en trouvant la porte de sortie vers un autre monde. Le level design présente un environnement labyrinthique, regorgant d'ennemies et de pièges en tout genre. Par cette idée, je voulais mettre l'accent sur les graphismes colorés.

Abyss

Abyss est un jeu 2D en vue de dessus permettant d'aller dans 4 directions.

Le jeu raconte l'histoire d'une jeune fille essayant de retrouver son ami perdu. Elle doit pour cela traverser les rues de Tokyo, les égouts pour enfin finir par les catacombes.

Le jeu comprend ainsi 3 niveaux dans lesquels il faut progresser. La façon de jouer est inspirée du jeu Undertale avec son système original de combat avec les ennemis et cette vue de dessus.

Le jeu a été développé sur la plateforme Unity au sein d'un groupe de 4 personnes. Mon travail a été de réaliser l'intégralité des assets d'environnements, de personnages avec leurs animations et des assets nécéssaires aux UI tandis que le reste du groupe s'occupait de la programmation du jeu.

Les assets ont été dessiné sur le logiciel Krita puis je les ai répartie sur Affinity Designer afin de créer les tilemaps
